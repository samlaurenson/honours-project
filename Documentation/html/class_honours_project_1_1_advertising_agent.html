Detailed Description

Class for the Advertising agent.

This agent is responsible for notifying house agents of slots that are currently available for exchange. Once this agent receives a request for an exchange, the advertiser agent will notify the agent with the desired slot of the request, where the decision will be made to accept or deny the exchange.
